SBB Showcase

Java • SQL • HTML • JavaScript • Swiss Railways

An interactive web application built with Java and HTML to showcase the passenger train fleet of the Swiss Federal Railways (SBB). It provides a visual and structured way to explore SBB’s rolling stock, from high-speed trains like the Giruno to regional workhorses like the FLIRT.

Features

  • Visual overview of SBB passenger trains
  • Train details such as type, service speed, and classification
  • Organized gallery-style presentation
  • Easy to extend with new models
